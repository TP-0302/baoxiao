============================================
  出差报销验算工具 - 手机版 (PWA)
============================================

【方式一：电脑开服务，手机浏览器访问（最简单）】

1. 双击运行 "启动服务器.bat"
2. 确保手机和电脑在同一个 WiFi 下
3. 用手机浏览器访问终端显示的地址：
   例如：http://192.168.1.100:8080
4. 安卓用户：浏览器菜单 → "添加到主屏幕" 即可像 App 一样使用
5. 苹果用户：Safari 打开 → 分享按钮 → "添加到主屏幕"

【方式二：部署到公网（推荐长期使用）】

免费方案 - GitHub Pages：
  1. 注册 GitHub 账号
  2. 创建仓库，上传 pwa 文件夹里的 3 个文件
  3. Settings → Pages → 启用，获得 https://xxx.github.io 地址
  4. 手机浏览器打开，添加到主屏幕，完美安装成 App

【功能说明】

✓ 8 类费用：车票、住宿、打车、美团、货拉拉、物流寄件、高速费、餐补
✓ 多笔金额用英文逗号分隔（如：123.5,200）
✓ 无支出填"无"或留空
✓ 一键验算汇总
✓ 复制结果到剪贴板
✓ 分享到微信/钉钉等
✓ 导出 PDF 打印
✓ 支持离线使用（首次加载后可断网使用）

【文件说明】

index.html     - 主程序（手机网页版）
manifest.json  - PWA 配置文件（让浏览器识别为 App）
sw.js          - 离线缓存服务
