============================================
  出差报销验算工具 — 手机使用指南
============================================

【方式一】把文件发到手机上直接用（最简单，不需要电脑）

  把以下 3 个文件通过微信/QQ/数据线发到手机：
    - index.html
    - manifest.json
    - sw.js

  安卓手机：
    用"文件管理器"找到 index.html → 用浏览器打开 → 即可使用
    （部分浏览器如 Via、Edge 支持添加到桌面）

  苹果手机：
    用"文件"App 找到 index.html → 长按 → 共享 → Safari 打开
    （功能正常使用，但添加到桌面需要 HTTPS 部署）

  ★ 数据完全保存在手机本地，无需网络，无需电脑


【方式二】部署到公网（推荐，可安装成 App）

  A. GitHub Pages（免费）：
     1. 注册 github.com 账号
     2. 创建仓库，上传 pwa 文件夹中的 3 个文件
     3. Settings → Pages → 保存
     4. 获得 https://你的用户名.github.io/仓库名 地址
     5. 手机打开 → 添加到主屏幕

  B. Gitee Pages / 码云（国内更快）：
     1. 注册 gitee.com 账号
     2. 创建仓库，上传文件
     3. 服务 → Gitee Pages → 启动
     4. 手机打开 → 添加到主屏幕


【方式三】电脑开热点，手机连（临时用）

  双击"启动服务器.bat"，手机连同一 WiFi
  浏览器打开显示的 http://192.168.x.x:8080 地址即可
